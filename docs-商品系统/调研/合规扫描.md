# 商品系统合规扫描

## 扫描范围

商品系统涉及的核心业务行为：商品信息发布、成色/质检描述、奢侈品鉴定、消费电子翻新销售、多国定价与展示、产品安全。

---

## 1. 美国联邦层面

### 1.1 FTC 二手商品披露要求 — 必须遵守

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **禁止虚假新品标识** | 退货/翻新/二手商品不得标为"new"（FTC Act Section 5） | 商品状态字段必须准确标注，系统层面限制 |
| **准确描述成色** | 必须如实披露使用痕迹、损坏、修复历史 | 成色分级体系 + 瑕疵描述字段为必填 |
| **翻新品标注** | "refurbished"无法律定义，但必须准确描述实际状态 | 消费电子需明确标注翻新程度和检测项目 |
| **原厂vs第三方翻新** | 若展示原厂品牌，必须披露翻新方身份 | 翻新商信息字段（厂翻/第三方翻新） |

### 1.2 INFORM Consumers Act — 必须遵守

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **高量卖家信息披露** | 年销售额 >$20,000 的三方卖家需公开姓名/地址/联系方式 | 商家信息展示页面需支持，入驻时采集 |
| **违规处罚** | 每次违规 $53,088 罚款 | 必须在商家管理系统中强制执行 |

### 1.3 Magnuson-Moss Warranty Act — 必须遵守（如提供保修）

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **保修披露** | >$10 消费品提供保修时必须标注"Full"或"Limited" | 消费电子保修信息字段设计 |
| **保修内容清晰** | 保修范围、期限、维修方式必须明确告知 | 保修条款模板化，减少商家合规风险 |
| **非强制提供** | 法律不要求必须提供保修，但提供了就必须合规 | 可选保修，但填了就要合规 |

### 1.4 CPSC 产品安全 — 必须遵守

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **召回产品禁售** | 已被 CPSC 召回的产品不得销售（含二手） | 需对接 CPSC 召回数据库，上架/定期扫描 |
| **平台责任** | CPSC 已裁定平台（如 Amazon）为"distributor"，承担产品安全责任 | looply 作为平台有法律责任，不能完全免责 |
| **电子产品安全** | 电池/充电器等有专项安全要求 | 消费电子品类需额外安全检查字段 |
| **处罚** | 每次违规最高 $100,000 罚款 | 必须建立召回商品拦截机制 |

### 1.5 假冒商品法律风险 — 必须遵守

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **Lanham Act（商标法）** | 销售假冒商标商品可追诉 2-4 年 | 奢侈品鉴定流程是法律护城河 |
| **18 USC § 2320** | 故意销售假冒品：初犯最高 10 年监禁 + $2M 罚款 | 平台需建立鉴定机制 + 举报通道 |
| **平台免责条件** | 平台需证明已采取合理措施防止假冒品流通 | 鉴定流程 + 卖家协议 + 举报处理 SLA |

## 2. 美国州级法规

### 2.1 加州 — 建议遵守（前期必做）

| 法规 | 要求 | 对 looply 的影响 |
|------|------|-----------------|
| **Prop 65** | 含特定化学物质的产品需警告标识 | 部分电子产品/皮革制品可能涉及，需警告标签字段 |
| **Song-Beverly Act** | 零售商对消费品有隐含保修义务 | 二手品的隐含保修期限需法务确认 |
| **eWaste 回收法** | 部分电子设备有回收费要求 | 消费电子品类可能需加收回收费 |

### 2.2 各州销售税 — 必须遵守

| 要求 | 说明 |
|------|------|
| **经济联结（Economic Nexus）** | 多数州对远程卖家设有销售额门槛（通常 $100K 或 200 笔交易） |
| **平台代扣代缴** | 多数州的 marketplace facilitator 法要求平台代扣代缴 |
| **二手品税率** | 大多数州对二手品征收与新品相同的销售税，少数州有豁免 |

**对 looply 影响**：作为 marketplace facilitator，looply 需在大多数州代扣代缴销售税。这在登录注册合规扫描中已覆盖，商品系统需配合提供商品税务分类码。

## 3. 欧洲/EU（后续扩展市场）

### 3.1 EU Consumer Rights Directive — 必须遵守（进入欧洲时）

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **2 年法定保修** | 所有消费品（含二手）享有至少 2 年法定保修 | 二手品保修期可缩短至 1 年（需明确告知） |
| **14 天无理由退货** | 线上购买的消费品有 14 天冷静期 | 退货流程必须支持 |
| **二手品合规率低** | 2025 年 EU 调查：超过半数二手在线卖家不合规 | 进入欧洲前需系统性合规改造 |

### 3.2 GPSR（General Product Safety Regulation）— 必须遵守（进入欧洲时）

| 要求 | 说明 | 对 looply 的影响 |
|------|------|-----------------|
| **产品安全标识** | 所有产品必须安全、可标识、可追溯 | 商品信息需含制造商/进口商信息 |
| **保修信息展示（2026.9 起）** | 所有网站和门店必须展示法定保修提示（Harmonised Notice） | 商品详情页需新增法定保修通知模块 |

### 3.3 各国差异

| 国家 | 特殊要求 |
|------|---------|
| 法国 | 维修指数（repairability index）强制展示，部分电子产品 |
| 德国 | BattG 电池法：含电池产品有回收标识和注册要求 |
| 英国（非 EU） | 脱欧后保修规则不同，CRA 2015 适用，30 天退货权 |

## 4. 二手电商特有合规

### 4.1 奢侈品鉴定 — 建议遵守

| 维度 | 说明 |
|------|------|
| **无强制鉴定法规** | 美国无联邦法要求二手奢侈品必须经过第三方鉴定 |
| **行业惯例** | The RealReal / Vestiaire Collective 等均提供鉴定服务，已成为消费者预期 |
| **品牌诉讼风险** | 奢侈品牌可能对平台发起商标侵权诉讼（即使商品是正品） |
| **建议** | 前期可用第三方鉴定服务（如 Entrupy、Real Authentication），不必自建团队 |

### 4.2 消费电子特有要求 — 必须遵守

| 维度 | 说明 |
|------|------|
| **数据擦除** | 二手电子设备必须完成出厂重置，擦除个人数据 |
| **电池安全** | 锂电池产品有运输安全要求（DOT/IATA 规定） |
| **FCC 认证** | 电子设备需符合 FCC 标准（原厂已认证，翻新不改硬件无需重新认证） |

## 5. 风险等级汇总

| 等级 | 合规项 | 处理建议 |
|------|--------|---------|
| 🔴 **P0 红线** | FTC 二手品标注、CPSC 召回品禁售、假冒品拦截、INFORM 卖家信息披露 | MVP 必须实现 |
| 🟠 **P1 高优** | 保修信息合规（Magnuson-Moss）、销售税代扣、数据擦除证明 | MVP 建议实现 |
| 🟡 **P2 中优** | 加州 Prop 65 警告、奢侈品鉴定流程、eWaste 回收费 | V2 实现 |
| 🟢 **P3 后续** | EU GPSR 保修通知、法国维修指数、德国电池法 | 进入欧洲市场时实现 |

## 6. 待确认项（需法务确认）

1. **隐含保修**：二手商品在各州的隐含保修期限和免责条件
2. **平台责任边界**：作为 marketplace facilitator，平台对商品质量/安全的法律责任范围
3. **奢侈品品牌授权**：展示品牌 logo/商标时是否需要额外授权（First Sale Doctrine 通常保护，但需确认）
4. **电池运输合规**：跨州/跨境运输锂电池产品的具体要求
