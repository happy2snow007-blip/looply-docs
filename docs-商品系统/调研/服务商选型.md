# 商品系统服务商选型

## 外部依赖清单

商品系统涉及以下外部能力需求：

1. 奢侈品鉴定服务
2. CPSC 召回产品数据库
3. 图片存储/CDN
4. 多语言翻译/本地化

---

## 1. 奢侈品鉴定服务

### 需求背景
looply 包含奢侈品品类，假冒品风险高（全球假冒奢侈品市场规模约 $450B）。前期无需自建鉴定团队，可用第三方服务。

### 候选服务商

| 维度 | Entrupy | Legit Check | Real Authentication |
|------|---------|-------------|---------------------|
| **鉴定方式** | AI + 微观图像分析 | 人工专家（上传照片） | 人工专家（上传照片） |
| **准确率** | 99.1%（官方数据） | 未公开 | 未公开 |
| **支持品牌数** | 22 个奢侈品牌 + 球鞋 + 珠宝 | 550+ 品牌指南 | 多品牌 |
| **定价** | $139-1,049/月（$21-28/次） | 按次付费（约 $5-30/次） | 按次付费（约 $10-30/次） |
| **API 集成** | 支持 API + SaaS | 不支持 API | 不支持 API |
| **硬件要求** | 专用微观摄像设备 | 无（手机拍照即可） | 无（手机拍照即可） |
| **附加能力** | Fingerprinting（防退货欺诈） | 免费鉴别指南 | — |
| **适用场景** | 高量级平台自动化集成 | 低量级/按需鉴定 | 低量级/按需鉴定 |

### 建议

| 阶段 | 推荐方案 | 原因 |
|------|---------|------|
| **MVP** | Legit Check / Real Authentication（按次付费） | 量少时成本低，无硬件门槛 |
| **规模化** | Entrupy（API 集成） | 自动化 + 高准确率 + 防退货欺诈 |

**注意**：Entrupy 需要专用微观摄像设备，意味着商家或平台仓库需配备硬件。如果 looply 采用"商家自发货"模式，Entrupy 在前期不适用，需要商家端上传照片 + 人工鉴定方案。

## 2. CPSC 召回产品数据库

### 需求背景
合规红线要求：已被 CPSC 召回的产品不得在平台销售。需要在商品上架时自动比对。

### 方案

| 维度 | CPSC 官方 API | 第三方封装（Apify 等） |
|------|-------------|---------------------|
| **数据覆盖** | 1973 年至今全部召回数据 | 基于官方数据二次处理 |
| **格式** | JSON / XML | JSON |
| **费用** | 免费（公共 API，无需 API Key） | 按调用量付费 |
| **搜索能力** | 按标题/产品名/描述/UPC 搜索 | 支持关键词/日期/召回号搜索 |
| **更新频率** | 实时（官方发布即可查） | 取决于爬取频率 |
| **集成难度** | 低（REST API，标准 HTTP） | 低 |

### 建议

直接对接 **CPSC 官方 API**，免费且实时，无需第三方。

实现方式：
- 商品上架时：用产品名/品牌/UPC 查询召回数据库
- 定时扫描：每日/每周扫描已上架商品，比对新增召回
- 命中召回：自动下架 + 通知商家

## 3. 图片存储/CDN

### 需求背景
商品图片是核心资产，需要：高速全球分发、自动格式转换（WebP/AVIF）、尺寸裁切、水印。

### 候选服务商

| 维度 | Cloudinary | imgix | Cloudflare Images |
|------|-----------|-------|-------------------|
| **定价** | Free 起；付费 $89/月起 | $10/月起 | $5/月（100K 图存储）+ $1/100K 分发 |
| **AI 功能** | 丰富（背景移除、AI 标签、智能裁切） | 中等（人脸检测、水印） | 基础 |
| **自动格式** | WebP / AVIF | WebP / AVIF | WebP / AVIF |
| **CDN 网络** | Akamai / Fastly | 多 CDN | Cloudflare 全球网络 |
| **DAM（资产管理）** | 内建 | 无 | 无 |
| **视频支持** | 支持 | 有限 | 独立产品（Stream） |
| **变换能力** | 300+ 种 | 100+ 种 | 有限（variants） |
| **规模成本** | 较高 | 中等 | 最低 |
| **接入难度** | 中等（SDK 丰富） | 低（URL 参数） | 低（API 简单） |

### 建议

| 阶段 | 推荐方案 | 原因 |
|------|---------|------|
| **MVP** | Cloudflare Images | 成本最低，全球 CDN 覆盖好，接入简单 |
| **规模化** | Cloudinary | AI 能力强（商品图自动背景移除、智能裁切），DAM 管理方便 |

**注意**：如果 looply 技术栈已使用 Cloudflare（DNS/WAF 等），Images 是自然选择。此为技术决策，需技术团队确认。

## 4. 多语言翻译/本地化

### 需求背景
looply 多国销售（前期美国，后续欧洲），商品标题/描述需要多语言支持。

### 候选方案

| 维度 | Weglot | Lokalise | 自建（Google Translate API / DeepL API） |
|------|--------|---------|--------------------------------------|
| **定位** | 网站级翻译（前端层） | 开发团队本地化管理 | 翻译 API 按需调用 |
| **适用场景** | 整站多语言 | APP/Web 多语言文件管理 | 商品内容动态翻译 |
| **AI 翻译** | 支持（机器翻译 + 人工校对） | 支持（多 LLM 路由 + Custom Model） | 原生 API |
| **定价** | €190/月起（10 万词） | $120/月起 | Google: $20/百万字符；DeepL: $5.49/百万字符 |
| **API 集成** | 有 | 有（CI/CD 集成好） | 原生 API |
| **电商适配** | 好（Shopify/WooCommerce 插件） | 中等（通用本地化） | 需自建翻译工作流 |
| **SEO** | 支持（hreflang 自动生成） | 需开发实现 | 需开发实现 |

### 建议

| 阶段 | 推荐方案 | 原因 |
|------|---------|------|
| **MVP（仅美国英语）** | 不需要翻译服务 | 前期单一市场 |
| **扩展欧洲** | DeepL API（商品翻译）+ Lokalise（UI 本地化） | DeepL 欧洲语言质量最优且成本低；Lokalise 管理 UI 多语言文件 |

**注意**：商品翻译和 UI 翻译是两个场景。商品标题/描述属于 UGC，适合 API 自动翻译 + 人工校对；UI 界面文案适合本地化管理平台。

## 汇总

| 外部依赖 | MVP 推荐 | 规模化推荐 | MVP 预估月成本 |
|---------|---------|-----------|--------------|
| 奢侈品鉴定 | Legit Check（按次） | Entrupy（API） | ~$0-300（按量） |
| CPSC 召回库 | CPSC 官方 API | CPSC 官方 API | $0（免费） |
| 图片 CDN | Cloudflare Images | Cloudinary | ~$5-50 |
| 多语言翻译 | 不需要（仅美国） | DeepL + Lokalise | $0（前期） |

**总计 MVP 阶段外部依赖成本**：约 $5-350/月（取决于鉴定量）
