查询逻辑示例：
SELECT product.* FROM product JOIN product_attribute pa ON product.id = pa.product_id JOIN attribute_value_mapping avm ON pa.attribute_id = avm.source_attribute_id AND pa.option_id = avm.source_option_id WHERE avm.target_option_id = '黑色系的option_id'
